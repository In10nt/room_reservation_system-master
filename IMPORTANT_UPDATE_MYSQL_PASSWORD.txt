⚠️ IMPORTANT: UPDATE YOUR MYSQL PASSWORD ⚠️

Before running the backend, you MUST update the MySQL password!

1. Open this file:
   backend/src/main/resources/application.properties

2. Find this line:
   spring.datasource.password=password

3. Replace "password" with your actual MySQL root password

4. Save the file

5. Then restart the backend

Example:
If your MySQL password is "mypass123", change it to:
   spring.datasource.password=mypass123

After updating, run:
   START_BACKEND.bat
